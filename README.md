# Sales Analytics Dashboard

A full-stack Sales Analytics Dashboard built using React, Flask, PostgreSQL, and Python for analyzing business sales performance and generating actionable insights from sales data.

## Features

* Interactive dashboard with KPIs
* Monthly sales trend analysis
* Region-wise sales visualization
* Top-selling products analysis
* Profit and revenue analytics
* REST API integration
* Responsive frontend UI
* Data cleaning and exploratory data analysis (EDA)
* Machine learning based sales forecasting (planned)

## Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Recharts

### Backend

* Flask
* REST APIs
* SQLAlchemy

### Database

* PostgreSQL

### Data Analytics & ML

* Pandas
* NumPy
* Matplotlib
* Scikit-learn

## Project Goals

This project aims to:

* Analyze sales data efficiently
* Visualize business KPIs
* Provide meaningful business insights
* Practice full-stack development and data analytics skills
* Build a portfolio-ready real-world project

## Dataset

The project uses the Superstore Sales Dataset containing:

* Orders
* Sales
* Profit
* Customer information
* Product categories
* Regional sales data
